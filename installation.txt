Helper file to install required packages.

Running JOFSTO requires:
    - pytorch
    - numpy

The code can run on GPU, or on the CPU only (much slower).
Please see [link](https://pytorch.org/) for PyTorch installation details.

We used python=3.6.13, pytorch==1.4.0, numpy==1.19.4, running on GPU with cuda 10.1, gcc 8.3.

Examples in our code may require additional packages, e.g. for simuating data.

Multiple options for creating environments, please see link for details:
    - Use conda [link](https://docs.conda.io) e.g. $ conda create -n ED_MRI; conda install torch numpy
    - Use pip [link](https://pip.pypa.io) e.g. $ pip install -r numpy torch
    - Use pyenv [link](https://github.com/pyenv)
